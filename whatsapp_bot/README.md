📱 WhatsApp Weather & Reminder Bot 🌦️⏰
*******************************************

A Python-based WhatsApp bot that provides real-time weather updates and allows users to set reminders. This bot is built using Twilio API for WhatsApp messaging and ngrok for exposing the local server to the internet.

🚀 Features :
*********************

🌤️ Get real-time weather updates for any city
⏰ Set reminders with custom messages
📲 Works directly on WhatsApp
🔄 Instant responses using webhook integration
🧠 Simple command-based interaction

🛠️ Tech Stack :
*******************

Python 3.x
Flask (for webhook server)
Twilio API (WhatsApp messaging)
ngrok (public URL tunneling)
OpenWeather API (for weather data)
